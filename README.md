# Ansible Role `volta`

Role that installs Volta and a default version of Node.

- <https://github.com/volta-cli/volta>
- <https://volta.sh/>

## FAQ

### How to bump the version of Volta?

Whenever this role is executed, Volta is updated to the latest version
automatically.
